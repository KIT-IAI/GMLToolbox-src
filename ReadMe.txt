========================================================================
    ANWENDUNG: BPlanGMLViewer-Projektübersicht
========================================================================

Der Anwendungs-Assistent hat die XPlanGMLToolbox-Anwendung erstellt. 

Die Datei enthält eine Zusammenfassung des Inhalts der Dateien für die 
XPlanGMLToolbox-Anwendung.

XPlanGMLToolbox.vcproj
    Dies ist die Hauptprojektdatei für VC++-Projekte, die mit dem Anwendungs-Assistenten
    erstellt werden. Sie enthält Informationen über die Version von Visual C++, mit der 
    die Datei generiert wurde, sowie über die Plattformen, Konfigurationen und Projektfeatures,
    die mit dem Anwendungs-Assistenten ausgewählt wurden.

Form1.cpp
   Dies ist die Hauptquelldatei der Anwendung.
    Enthält den Code zur Anzeige des Formulars.

Form1.h
    Enthält die Implementierung der Formularklasse und die InitializeComponent()-Funktion.

AssemblyInfo.cpp
    Enthält benutzerdefinierte Attribute zum Ändern der Assembly-Metadaten.

/////////////////////////////////////////////////////////////////////////////
Weitere Standarddateien:

StdAfx.h, StdAfx.cpp
    Werden zum Erstellen einer vorkompilierten Headerdatei (PCH) 
    namens 'Win32.pch' und einer vorkompilierten Typendatei namens 'StdAfx.obj' verwendet.
